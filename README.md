## 🚀 快速开始
### 安装指南
```bash
# 基础功能安装
pip install flyf

# 完整版（含WebUI）
pip install flyf[all]
```

### 3行代码体验
```python
from flyf import iChat

assistant = FlyAI(api_key="free_trial_key")  # 使用内置测试密钥
print(assistant.chat("用Python实现快速排序"))
```

### 启动Web界面
```bash
flyf web --port 8000
```
访问 ➡️ `http://localhost:8000` 开启可视化交互
